Comandos Básicos do Ubuntu

  • ls - Lista arquivos e pastas no diretório atual.
  • cd - Muda o diretório de trabalho.
  • pwd - Exibe o caminho completo do diretório atual.
  • sudo apt update - Atualiza a lista de pacotes disponíveis.
  • sudo apt upgrade - Atualiza os pacotes instalados no sistema.
  • mkdir - Cria um novo diretório.
  • rm - Remove arquivos ou diretórios.

Esses comandos são fundamentais para quem está começando a usar o Ubuntu. Existem muitos outros comandos, e você pode consultar a documentação oficial para aprender mais.
